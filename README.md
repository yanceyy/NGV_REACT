# NGV-react-TS

Rewrite the [NGV](https://www.ngv.vic.gov.au/) using modern front end techniques.

## Usage
* Install dependencies
```bash
yarn install
```
* Run the project in development mode
```bash
yarn start
```
* Build the project
```bash
yarn build
```
